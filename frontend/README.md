# Currency Converter - Frontend

This is the **Angular 19 frontend** for the Currency Converter project. It connects to a NestJS backend to fetch real-time and historical currency rates.

## Features

- Convert between any supported currencies dynamically
- Historical exchange rates (select a date)
- Real-time conversion history stored in `localStorage`
- Mobile-first responsive UI
- Loaders shown during API requests
- Swap currencies easily

## Tech Stack

- Angular 19
- Angular Material
- TypeScript
- RxJS
